WORDPRESS API (PRODUCTIE)

Alle dashboard-endpoints staan live op materialdistrict.com (batches 1 t/m 4):

- Batch 1: profile, brand profile, materials list + status toggle
- Batch 2: portal, requests, interactions, lead-routing, statistics
- Batch 3: bookmarks, boards, saved searches, insider insights, user invoices, material form CRUD
- Batch 3+: GET /md/v2/dashboard/material-categories (alleen leaf terms, shape { id, l1, l2, l3 })
- Batch 4: featured (Partner+), brand invoices, brand delete, brand-candidates / claim / request-new

Auth blijft hetzelfde: Authorization: Bearer <JWT> via /md/v2/auth/login. Brand-routes gebruiken numeriek brandId; in de UI map je via user.brands[] (slug ↔ id).
