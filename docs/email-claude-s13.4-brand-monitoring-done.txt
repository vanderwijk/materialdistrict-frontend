== Plugin (master) — rest-dashboard-batch2.php ==

1. LEAD ROUTING — GET + POST /md/v2/dashboard/brands/{id}/lead-routing

   Response + POST body keys (exact snake_case, conform jouw contract):
     restrict_to_listed_countries
     sample_requests_insiders_only
     downloads_insiders_only

   Persisted as brand post meta:
     _brand_restrict_to_listed_countries
     _brand_sample_requests_insiders_only
     _brand_downloads_insiders_only

   Waarden als 'true'/'false' strings; gelezen via md_meta_truthy().
   Default false als nog niet gezet. POST accepteert partial body — booleans
   worden opgeslagen ook als routes niet meegestuurd worden.

2. STATISTICS — GET /md/v2/dashboard/brands/{id}/statistics

   Nieuw veld:
     brochures: [ { title, downloads } ]

   Implementatie md_dashboard_brand_brochure_stat_rows():
     • Primair: items uit _brand_dashboard_downloads (dashboard brand profile)
     • Fallback: legacy PDF child attachments op brand (zelfde patroon als
       theme page-statistics.php)
     • downloads per item: attachment meta _brochure_downloads /
       brochure_downloads / download_count

   Lege array als brand geen brochures heeft — Brochures-tab toont lege state.
