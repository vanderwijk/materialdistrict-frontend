Hosting — geen tweede DB op WP Engine
-------------------------------------

WP Engine laat geen aparte analytics-DB toe op hetzelfde account. Plan:

| Fase | Keuze |
|------|--------|
| Nu | AWS RDS of Aurora MySQL (eu-central-1), alleen poort 3306 open voor WP Engine outbound IP's |
| Lokaal | Fallback wp_mdanalytics_* op hoofd-DB (geen extra infra) |
| Later bij volume | ClickHouse; zelfde REST/read-contract, andere engine achter de PHP-laag |

Plugin-config via MD_ANALYTICS_DB_* in wp-config.php — geen code-wijziging aan jouw kant.
