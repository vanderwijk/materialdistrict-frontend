## Symptoom

/dashboard/brands/{slug}/featured crasht met:

  RangeError: Invalid time value

Console: [dashboard] render error → dashboard/error.tsx (“Something went wrong”).

Materials-pagina van hetzelfde merk laadt wél.

---

## Oorzaak

WordPress levert `featured_slots_reset_date` als volledige ISO-datetime, bv.:

  2027-06-03T13:00:49+00:00

In FeaturedPanel werd (impliciet) geformatteerd met:

  new Date(`${iso}T00:00:00`)

Dat wordt een ongeldige string (`…+00:00T00:00:00`). `Intl.DateTimeFormat().format()` gooit dan RangeError.

---

## Fix — 3 bestanden

### 1) src/components/dashboard/panels/FeaturedPanel.tsx

Vervang de datum-helpers boven de component door een veilige parser die alleen YYYY-MM-DD gebruikt:

- Nieuwe helper `dateFromIso(iso: string): Date | null`:
  - `const day = iso.slice(0, 10)`
  - regex-check `^\d{4}-\d{2}-\d{2}$`
  - `new Date(\`${day}T12:00:00\`)` + NaN-check

- `fmtDay`, `fmtWeekRange`, `fmtResetDate` gebruiken alleen `dateFromIso`; bij null → `'—'`

- Wrap `Intl.DateTimeFormat(...).format(d)` in try/catch → bij fout `'—'`

Verwijder elke `new Date(\`${iso}T00:00:00\`)` waar `iso` de ruwe WP-reset-string kan zijn.

### 2) src/lib/dashboard/data.ts

Bij `getFeaturedSlots`, vóór de slot-mapper:

```ts
function normalizeDashboardDate(iso: string | null | undefined): string | null {
  if (!iso) return null
  const day = String(iso).slice(0, 10)
  return /^\d{4}-\d{2}-\d{2}$/.test(day) ? day : null
}
```

In de return van `getFeaturedSlots`:

```ts
resetDate: normalizeDashboardDate(raw.featured_slots_reset_date),
```

(niet meer raw `featured_slots_reset_date` direct doorgeven)

Optioneel in dezelfde file — `getBrandMaterials`: try/catch zoals andere panels; 403/404 → `[]`; bearer via `requireToken()`.

### 3) src/app/dashboard/brands/[brandSlug]/featured/page.tsx (aanbevolen)

Laad featured + materials in try/catch. Bij `DashboardApiError` toon een `EmptyState` met concrete fout (code + HTTP status) i.p.v. de generieke dashboard error boundary.

Imports o.a.: `DashboardApiError`, `EmptyState`, `IconAlert`, type `FeaturedSlotsData`.
